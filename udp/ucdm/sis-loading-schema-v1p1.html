SIS Loading schema v. 1.1

Proposed changes


Proposed changes

Version 1.1 of the loading schema is intended to create new models and extend existing ones in order to support a broader swath of teaching & learning data, including data from Unizin Engage and Unizin Order Tool.

New models

Session

A session defines a period of instruction in a given term.

Data element Header Description
SIS Session ID SisIntId The internal primary key used by the SIS to define a session record.
SIS Term ID TermId The foreign key to the Academic term to which the Session belongs.
Name Name

The name of the Session. The value is typically intended to be humand-readable ("First 8-week session") and, in concert with an Academic term name, fully communicate when a Course section is offered ("Fall 2018 First 8-week session").
Instruction start date StartDate

The beginning of instruction for Course sections in the Session.
Instruction end date EndDate

The end of instruction for Course sections in the Session.
Inclusive access bill after date BillAfterDate

The date after which inclusive access content can be billed to the institution.
top

Campus

Some institutions are system institutions with multiple Campuses. These may include "virtual campuses" that represent online programs, which is common to many Unizin Members.

Data element Header Description
SIS Campus ID SisIntId The internal primary key used by the SIS to define a Campus record.
Name Name The full name of the Campus in the Institution's system (e.g., "Fort Wayne").
Abbreviation Abbreviation

The abbreviation for the campus name. The intent is to capture the shorthand used in schedule of classes and other administravia to refer to the campus (e.g., "BL" for Bloomington).
top

Academic organizations

Schools, Programs, Colleges, and Departments are Academic organizations responsible for delivering instruction. A particular instance of an Academic organization is unique to a campus. However, they may be multiple instances of Academic organizations that share the same Organization Code.

While a campus may only have a single Department of Sociology, for example, a system Institution likely has many Departments of Sociology, each with a unique SIS Academic organization ID but with a common Organization Code.

Data element Header Description
SIS Academic organization ID SisIntId The internal primary key used by the SIS to define an academic organization record.
SIS Campus ID CampusId The foreign key to the Campus to which the Academic organization belongs.
Parent Academic organization ParentId

(optional) The foreign key to the parent Academic organization of this Academic organization.
Academic organization code Code

A code for the Academic organization. The intended value is the shorthand used in schedule of classes and other administravia to refer to the Academic organization (e.g., "CAS").
Name Name

The humand-readable name of the Academic organization (e.g., "The College of Arts and Sciences").
top

Course grade

The grade a person receives for a course.

While this field will likely be updated by Canvas during a session, the SIS ultimately owns final, end-of-term grades.

Data element Header Description
SIS Academic organization ID SisIntId The internal primary key used by the SIS to define an academic organization record.
SIS Campus ID CampusId The foreign key to the Campus to which the Academic organization belongs.
Parent Academic organization ParentId

(optional) The foreign key to the parent Academic organization of this Academic organization.
Academic organization code Code

A code for the Academic organization. The intended value is the shorthand used in schedule of classes and other administravia to refer to the Academic organization (e.g., "CAS").
Name Name

The humand-readable name of the Academic organization (e.g., "The College of Arts and Sciences").

Changes to existing models

Academic term

Data element Header Status Description
Instruction start date InstrBeginDate Remove This value is replaced by a Session's instruction start date.
Instruction end date InstrEndDate Remove This value is replaced by a Session's instruction end date.
top

Person

Data element Header Status Description
Federated identity management identifier FedIdentity New The unique identifier for an individual released by an Institution's Identity Provider, typically in an SSO scenario. In Shibboleth, this value will usually correspond to the eduPersonTargetedId (EPPN).
Varsity athlete indicator VarsityAthlete New If true, indicates that the student is a varsity athlete at the institution. If false, indicates that the student is not a varsity athlete at the institution.
Academic career Career New A code representing a grouping of academic programs for statistical purposes. An academic career exists for each school/college by the program level offered. This will take an option set.

Option set values: Dental, Graduate, Undergraduate, Law, Medical, Optical.
Major Major New The student's major course of study.
Minor Minor New The student's minor course of study.
City residency City New The city in which the student is a resident.
State residency State New The state in which the student is a resident.
AdmittanceTerm AdmitTermId New The SIS term ID for the Academic term to which the student was initially admitted.
First-generation student FirstGeneration New If true, the student is the first in their family to attend an institution of higher education in the United States. If false, the student is not the first in their family to attend an institution of higher education in the United States.
Legal sex Sex Updated The legal sex of a person.
Gender identity Gender New The gender to which the individual self-identifies.
Gender pronouns Pronoun New The set of gender pronouns by which the individual prefers to be addressed. This will take an option set.

Financial aid data will also be ingested, but the particular elements that will comprise that data is still under investigation.

Course section

Data element Header Status Description
SIS session ID SessionId New The foreign key to the Session in which the Course section is delivered.
SIS academic organization ID AcademicOrgId New The foreign key to the Academic organization responsible for delivering this Course section.
Class number Class New A unique number for the Course section. This may or may not be the same as the internal SIS Course section identifier.
Section number Section New Among the Course sections grouped by a particular Course offering, this is a unique identifier. Typical examples include "001", "002", "003", etc.
Section type SectionType New The type of instruction delivered in this section. Institutions may refer to a section type as a "component" of a Course offering. This will take an option set.

Option set values: Lecture, IndependentStudy, Laboratory, Research, Discussion, Activity, Internship, Reading, Practicum, Seminar, Clinic, Recitation, Drill, Studio, Practice, Field, Methods, Colloquium, Demo
SIS term ID TermId Remove Course sections are delivered in a Session, rather than an Academic term.
