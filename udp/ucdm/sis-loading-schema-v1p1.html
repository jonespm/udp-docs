SIS Loading schema v. 1.1

Proposed changes


Proposed changes

The SIS loading schema v1.1 introduces new entities and entity changes to the UCDM. There are two major goals of version 1.1. First, it enriches the Person model with data pertinent to learning analytics research. Second, it adds new concepts that better-model Term-based academic processes.

New models

Session

A Session is an administrative unit that defines an actual instructional period during a term. A session is always unique to a Term and will have many Course offerings.

Data element Header Description
SIS Internal Session ID SisIntId The internal primary key used by the SIS to define a Session record.
SIS External Session ID SisExtId The unique, global ID for a Session generated in your institutional SIS for use by external tools such as Canvas, LTI applications, etc.
SIS Internal Term ID TermId The foreign key to the Academic term to which the Session belongs.
Name Name

The name of the Session. The value is typically intended to be humand-readable ("First 8-week") and, in concert with an Academic term name, fully communicate when a Course section is offered ("Fall 2018 First 8-week").
Instruction begin date InstrBeginDate

The beginning of instruction for Course sections in the Session.
Instruction end date InstrEndDate

The end of instruction for Course sections in the Session.
Inclusive access bill after date InclusiveAccessBillAfterDate

The date after which inclusive access course materials can be billed to the institution.

Campus

A Campus is the collection of departments, colleges, schools, and buildings that belong to a particular institution. All Universities have at least one campus. Some Universities are systems with multiple Campuses. These may include "virtual campuses" that represent online programs, which is common to many Unizin Members.

Data element Header Description
SIS Internal Campus ID SisIntId The internal primary key used by the SIS to define a Campus record.
SIS External Campus ID SisExtId The unique, global ID for a Campus generated in your institutional SIS for use by external tools such as Canvas, LTI applications, etc.
Name Name The full name of the Campus in the Institution's system (e.g., "Fort Wayne").
Abbreviation Abbreviation

The abbreviation for the campus name. The intent is to capture the shorthand used in schedule of classes and other administravia to refer to the campus (e.g., "BL" for Bloomington).
top

Academic organizations

An Academic organization is a School, Program, College, or Department responsible for delivering instruction. A particular Academic organization is always unique to a campus. However, Academic organizations across different campuses may share the same Organization Code. For example, there may exist a Department of Sociology at multiple campuses of a University system. Each instance can be uniquely identified but all may share the same Academic organization code.

Data element Header Description
SIS Internal Academic organization ID SisIntId The internal primary key used by the SIS to define an academic organization record.
SIS External Academic organization ID SisExtId The unique, global ID for an Academic organization generated in your institutional SIS for use by external tools such as Canvas, LTI applications, etc.
SIS Internal Campus Internal ID CampusId The foreign key to the Campus to which the Academic organization belongs.
SIS Internal Parent Academic organization ID ParentId

(optional) The foreign key to the parent Academic organization of this Academic organization.
Academic organization code Code

A code for the Academic organization. The intended value is the shorthand used in schedule of classes and other administravia to refer to the Academic organization (e.g., "CAS").
Name Name

The human-readable name of the Academic organization (e.g., "The College of Arts and Sciences").

Course grade

The grade a person receives for a course.

While this field will likely be updated by Canvas during a session, the SIS ultimately owns final, end-of-term grades.

Data element Header Description
SIS Internal Course grade ID SisIntId The internal primary key used by the SIS to define an academic organization record.
SIS External Course grade ID SisExtId The unique, global ID for a Course grade generated in your institutional SIS for use by external tools such as Canvas, LTI applications, etc.
SIS Internal Course section ID CourseSectionId The foreign key to the Course section to which the Course grade belongs.
SIS Internal Person ID PersonId

The foreign key to the Person to whom this Course grade belongs.
Score Score

The absolute score for the course grade. Often, this will be a numerical score ranging from 0-100 (but it need not be). The intent is to capture a raw version of a course grade distinct from that reported in formal records.
Published Score PublishedScore

The published score for the Course grade. Often, this will be the letter version of the Score. The intent is to capture the formal, published score on record.
Workflow State WorkflowState

The state of the course grade, which is likely to change over the course of the term. This will take an option set. Likely values include "Incomplete," "Completed," and "Withdrew."

Person

Data element Header Status Description
Federated identity management identifier FedIdentity New The unique identifier for an individual released by an Institution's Identity Provider, typically in an SSO scenario. In Shibboleth, this value will usually correspond to the eduPersonTargetedId (EPPN).
Varsity athlete indicator VarsityAthlete New If true, indicates that the student is a varsity athlete at the institution. If false, indicates that the student is not a varsity athlete at the institution.
Academic career Career New A code representing a grouping of academic programs for statistical purposes. An academic career exists for each school/college by the program level offered. This will take an option set.

Option set values: Dental, Graduate, Undergraduate, Law, Medical, Optical.
Major Major New The student's major course of study.
Minor Minor New The student's minor course of study.
City residency City New The city in which the student is a resident.
State residency State New The state in which the student is a resident.
AdmittanceTerm AdmitTermId New The SIS term ID for the Academic term to which the student was initially admitted.
First-generation student FirstGeneration New If true, the student is the first in their family to attend an institution of higher education in the United States. If false, the student is not the first in their family to attend an institution of higher education in the United States.
Legal sex Sex Updated The legal sex of a person.
Gender identity Gender New The gender to which the individual self-identifies. (Note: an option set will be forthcoming. At present, we assume that Institutions do not yet systematically gather gender identity assertions. We expect the value to be null for some time.)

Financial aid data will also be ingested, but the particular elements that will comprise that data is still under investigation.

Course offering

Data element Header Status Description
SIS Internal Campus Internal ID CampusId New The foreign key to the Campus where the Course offering is delivered.
SIS Internal Session ID SessionId New The foreign key to the Session to which the Course offering belongs.
Academic career Career New A code representing a grouping of academic programs for statistical purposes. An academic career exists for each school/college by the program level offered. This will take an option set.

Option set values: Dental, Graduate, Undergraduate, Law, Medical, Optical.
SIS Internal Term ID TermId Remove Course offerings are offered in a given Session, which belongs to a specific Term. The course offering-to-term relationship can be derived via the session.
top

Course section

Data element Header Status Description
SIS academic organization ID AcademicOrgId New The foreign key to the Academic organization responsible for delivering this Course section.
Class number ClassNumber New A unique number for the Course section. This may or may not be the same as the internal SIS Course section identifier.
Section type SectionType New The type of instruction delivered in this section. Institutions may refer to a section type as a "component" of a Course offering. This will take an option set.

Option set values: Lecture, IndependentStudy, Laboratory, Research, Discussion, Activity, Internship, Reading, Practicum, Seminar, Clinic, Recitation, Drill, Studio, Practice, Field, Methods, Colloquium, Demo
SIS term ID TermId Remove A Course section belongs to a Course offering, which in turn belongs to a Session, which in turn belongs to a Term. Consequently there is no reason to model a Course section-Term relationship here.
